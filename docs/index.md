# BrainNotFound

Bienvenue dans la documentation de **BrainNotFound**, la plateforme d'évaluation intelligente avec correction par IA.

## Qu'est-ce que BrainNotFound ?

BrainNotFound est une application web open-source qui permet aux enseignants de :

- **Créer des quiz en Markdown** : Un format simple et lisible pour rédiger vos évaluations
- **Corriger automatiquement** : Les QCM sont corrigés instantanément, les questions ouvertes sont évaluées par Claude (IA d'Anthropic)
- **Suivre la progression** : Analytics détaillés pour identifier les lacunes et accompagner vos étudiants
- **Gérer plusieurs organisations** : Architecture multi-organisations pour les écoles et entreprises

## Fonctionnalités principales

| Fonctionnalité | Description |
|----------------|-------------|
| Quiz Markdown | Créez des quiz avec une syntaxe simple et intuitive |
| Correction IA | Questions ouvertes évaluées par Claude avec feedback personnalisé |
| Entretiens IA | Simulez des entretiens avec des personnages IA pour évaluer les soft skills |
| Multi-organisations | Gérez plusieurs organisations sur une seule instance |
| Groupes | Organisez vos étudiants par classe ou projet |
| Temporisation | Définissez des limites de temps et périodes de disponibilité |
| Analytics | Suivez les résultats et exportez les données |
| Multilingue | Interface et feedback IA en français et anglais |

## Versions disponibles

### Community (Gratuit)

- Auto-hébergement sur votre infrastructure
- Code source ouvert (GPL v3)
- Toutes les fonctionnalités de base
- Correction IA illimitée (avec votre clé API Anthropic)
- Support communautaire via GitHub

### SaaS (Sur devis)

- Hébergement et maintenance inclus
- Correction IA incluse (pas besoin de clé API)
- Support prioritaire
- Mises à jour automatiques
- Analytics avancés et exports
